A Timestamp Microservice

How-to:

  • Pass a string as a parameter (example: January 1, 2016)
  • If it contains either a unix timestamp or a natural language date, returns both the Unix timestamp and the natural language form of that date.
  • If it does not contain a date or Unix timestamp, it returns null for those properties.

Example usage:

https://historical-feels.glitch.me/December%2015,%202015

https://historical-feels.glitch.me/1450137600

Example output:

{ "unix": 1450137600, "natural": "December 15, 2015" }
